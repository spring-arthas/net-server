# 云盘目录删除同步修复

## 目标

- macOS 目录树删除操作必须显示确认提示，并真正发送目录删除请求。
- 服务端删除前递归检查选中目录及其子目录中的有效文件。
- 存在 `del = N`、`is_file = Y`、`is_exist = Y` 的文件时拒绝删除，并返回明确原因。
- 不存在有效文件时，删除选中目录下的文件系统目录树，并逻辑删除对应的数据库目录记录。
- 校验目录属于当前登录用户，禁止删除用户根目录和系统保护目录。
- 删除成功后刷新目录树，并清理已经失效的选中、展开状态。

## 验证场景

1. 有有效文件的目录删除失败，数据库和文件系统均保留。
2. 只有空子目录的目录可以删除，子目录数据库记录和文件系统目录同步删除。
3. 只有已删除或无效文件记录的目录可以删除。
4. 删除不存在、已删除、非当前用户或根目录时返回明确错误。
5. 前端取消确认不发送请求；成功和失败均正确恢复按钮状态并展示结果。
